# Evidence Guide

## Best practice
A dedicated `assets/` folder keeps screenshots that prove all my steps, like:
- successful SSO / identity check
- bucket creation
- EC2 launch and running status
- PostgreSQL service active
- DBeaver connection via SSH
- Apache page reachable
- CloudWatch alarm / SNS setup
- Redshift table creation or query result
- Aurora SSH tunnel / connection proof
- DynamoDB batch-get result
